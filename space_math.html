Math Formatting Tool With the Shallowest Learning Curve

"Think of the ways people type math when they are texting." Space Math intends to simplify math-inputting and support screen readers without ambiguity. Our core idea is to use space to determine the grouping of things. For example, $1+2 / 3$ and $1 + 2 / 3$ will produce outputs with different groupings. The first one means (1 + 2) / 3, the second one means 1 + (2 / 3). The program now supports Chinese inputs, making it easy for Chinese people who do not know English to format math. Future plans include working with MathJax to implement screen reader capability. Ambiguity like absolute value and cardinality will be addressed to ensure precise screen reader outputs.Disclaimer: this user interface is for testing only. The launch version's code will become open source and available for everyone to use.
